aiSound

Animated Interactive Sound for Web apps

aiSound is essentially a collection of synthlets - individual specialized interactive synthesizers that you can use in your web apps.

Here is the (so far very short) list of aiSounds for your exploration and exploitation.

The Sound Model API for Web Developers:

Just import the sound factory, create an instance of the sound, then use the aiSound API to make some noise:
  • play()
  • release()
  • stop()

  • setParam([name -or- number],val)
  • setParamNorm([name -or- number], val)
And if you need info, you can also use:
  • getParam([name, number], ["name", "type", "val", "normval", "min","max"])
  • getNumParams()
  • getAboutText();

The simplicity and consistency makes sounds models more shareable, and enables clean, readable, generalizable, and reusable code.

FM Mouser is an example that shows how easy interactive sound can be for javascript-coding web developers.
